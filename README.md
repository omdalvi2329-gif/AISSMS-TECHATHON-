# AgriSetu - Smart Farming Platform

A modern, farmer-focused agri-tech web application designed for hackathon presentation.

## Features

### Login Page
- **Mobile-First Design**: Optimized for farmers using mobile devices
- **Multi-Language Support**: English, Hindi (हिंदी), and Marathi (मराठी)
- **Premium UI**: Black and green color scheme with modern glassmorphism effects
- **Simple Authentication**: Name and mobile number only (no password/OTP for MVP)
- **Form Validation**: Real-time validation with user-friendly error messages
- **Responsive Design**: Works seamlessly on all device sizes

## Technology Stack

- **React 18**: Modern React with hooks
- **CSS3**: Advanced styling with gradients, animations, and backdrop filters
- **Mobile-First**: Responsive design approach
- **Component-Based**: Clean, maintainable code structure

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd AgriSetu
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
